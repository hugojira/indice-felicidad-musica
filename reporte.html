Introduccin

En este proyecto se busca relacionar el ndice de felicidad mundial con la las propiedades de audio de las canciones mas escuchadas en los distintos pases. Para esto, se usar los datos oficiales del World Happiness Report para el ndice de felicidad, y de Spotify para las canciones ms escuchadas y sus caractersticas.

Se buscan investigar los siguientes puntos

  • Hay una correlacin entre el ndice de felicidad y la positividad/negatividad en las canciones ms populares?
  • Se puede diferenciar un patrn en las canciones populares de un pas en base a su ndice de felicidad?

Obtencin de datos

Se trabajo con 2 tipo de datos, una fue el ndice de felicidad mundial para el ao 2020 y la otra caractersticas de audio de canciones

ndice de felicidad

Se obtuvieron los datos para el ao 2020, que son promedios de los aos 2018,2019 y 2020, para cada pas. Se descargaron de la fuente oficial de The World Happiness Report.

Caractersticas de audio

Para las caractersticas de audio se trabaj con datos proporcionados por la API Web de Spotify. Se buscaron las playlists de top 50 canciones ms escuchadas para cada pas y se hizo una consluta a la pgina para descargar las respectivas caractersticas de audio.

Se pudieron recaudar datos de canciones solamente para 65 de los 149 pases, por lo que en los anlisis de datos que se harn se toman en cuenta a 65 pases solamente.

Resultados

Categorizacin de variables

Se hizo una categorizacin en el ndice de felicidad en base al espaciamiento de los datos, las categoras elegidas fueron

  • bajo
  • medio
  • alto

Relaciones

Se graficaron diagramas de caja para ver si hay alguna separacin de las caractersticas de audio, respecto a las categoras del ndice de felicidad que se hicieron

Caractersticas de audio respecto al ndice de felicidad

Se ve que en base a las 3 categoras no se observa mucha diferencia en las caractersticas de audio a simple vista, por lo que esto sera adecuado investigarse con algn mtodo de clasificacin de aprendizaje automtico supervisado.

Adems, se hace una grfica del ndice de felicidad respecto a los promedios de valence de las 50 canciones ms escuchadas por cada pas. Se dividio en tres grupos con un algoritmo de agrupamiento de k-medias, lo que se distingue por colores en la grfica:

ndice de felicidad respecto a la positividad/negatividad de las canciones

En general el agrupamiento se observa bueno, hace una clara distincin en tres partes al eje y del ndice de felicidad. Lo que indicara que podra haber alguna relacin en la positivdad/negatividad de las canciones de acuerdo al ndice de felicidad.

Conclusiones

Se observa una relacin no lineal entre el ndice de felicidad y el promedio de valence de las 50 canciones ms escuchadas de cada pas, y se puede hacer una separacin por clusters de k-medias. Para este caso se vi que agrupar en 3 es adecuado.

Por ltimo, para involucrar a las dems caractersticas numricas restantes como energy o acousticness, lo que puede hacerse es aplicar algn algoritmo de clasificacin tomando como base las clases creadas para el ndice de felicidad, de bajo, medio, alto. Un algoritmo adecuado podra ser el de k vecinos ms cercanos o bien algn algoritmo estadstica de clasificacin o de inferencia con lgica difusa. Un algoritmo como redes neuronales artificiales no sera muy adecuado por los pocos datos que se tienen.
